# Antigravity Editor Integration for Unity

## How to install
<br>
- Unity->Window->Package Manager<br>
- Click "+" left corner<br>
- Add package from git URL<br>
- Insert your Antigravity package repository URL<br>
- Add<br>
- Done

## Features
- Automatic discovery of Antigravity Editor installations
- Generate C# project files for IntelliSense
- Open files at specific line numbers
- Workspace detection for running instances
